Tantangan terberatnya adalah membuat grafik ini responsif dan tidak menutupi teks. Solusinya?

1.  Menggunakan **Split View Layout** (Teks di kiri, Graph di kanan).
2.  Menambahkan **ResizeObserver** agar grafik selalu menyesuaikan ukuran layar secara real-time.

```tsx
// Cuplikan kode ResizeObserver untuk Graph
useEffect(() => {
  const resizeObserver = new ResizeObserver((entries) => {
    for (const entry of entries) {
      const { width, height } = entry.contentRect;
      if (width > 0 && height > 0) {
        setDimensions({ w: width, h: height });
      }
    }
  });
  // ...
}, []);
```
